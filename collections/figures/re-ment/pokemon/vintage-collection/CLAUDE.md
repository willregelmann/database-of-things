# Vintage Collection — curation hints

## "Type:" subtitling

Re-Ment's own product name for the debut release is "Pokémon VINTAGE
COLLECTION Type:Steel" — the series is subtitled by Pokémon type rather than
sequentially numbered. As of this curation (2026-07), no other "Type:"
release has appeared on either the active catalog
(`re-ment.co.jp/product/brand.php?c=pokemon`) or the discontinued-products
archive (`re-ment.co.jp/product/discontinue.php`) — Type:Steel is the only
release so far, filed as `vol-01`.

Whether a future release (e.g. a hypothetical "Type:Fire") becomes `vol-02`
of this same series or ships as its own sibling series is an open question —
decide it deliberately when curating one, rather than picking a placement to
unblock a single PR (same open-question pattern as Terrarium Collection's
EX/DX sub-lines — see `../terrarium-collection/CLAUDE.md`).
